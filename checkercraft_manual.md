# CheckerCraft

Minecraft Education Edition用のチェス盤（市松模様）作成プログラム

## 概要

CheckerCraftは、Minecraft内で美しいチェス盤パターンを自動生成するPythonプログラムです。標準的な8×8のチェス盤から、最大100×100の巨大なパターンまで作成できます。

## 特徴

- 🎯 **標準チェス盤**: デフォルトで8×8の本格的なチェス盤を作成
- 📏 **カスタムサイズ**: 5×5から100×100まで自由にサイズ指定可能
- 🎨 **美しい配色**: 白いコンクリートと黒いコンクリートで鮮やかなコントラスト
- 👀 **見やすい設計**: 各マス3×3ブロックで視認性抜群
- ⚡ **高速生成**: プレイヤーの足元から瞬時に作成
- 🛡️ **エラーハンドリング**: 無効な入力を適切に処理

## ファイル

- `checkercraft.py` - メインプログラム
- `checkercraft_manual.md` - このマニュアル

## 使い方

### 1. プログラムの読み込み

1. Minecraft Education Editionを起動
2. コードビルダーを開く（Cキー）
3. **Pythonモード**を選択
4. `checkercraft.py` を開く

プログラムを読み込むと、自動的に使い方メッセージが表示されます。

### 2. コマンド実行

チャットで以下のコマンドを入力：

#### 基本コマンド
```
checkercraft
```
デフォルトの8×8チェス盤（24×24ブロック）を作成

#### サイズ指定
```
checkercraft [サイズ]
```
指定したサイズのチェス盤を作成（5〜100の範囲）

### 3. 実行例

| コマンド | 結果 |
|---------|------|
| `checkercraft` | 8×8のチェス盤（24×24ブロック） |
| `checkercraft 5` | 5×5の小さなチェス盤（15×15ブロック） |
| `checkercraft 10` | 10×10の中サイズチェス盤（30×30ブロック） |
| `checkercraft 20` | 20×20の大型チェス盤（60×60ブロック） |
| `checkercraft 50` | 50×50の巨大チェス盤（150×150ブロック） |
| `checkercraft 100` | 100×100の超巨大チェス盤（300×300ブロック） |

## 仕様

### サイズ制限
- **最小サイズ**: 5×5
- **最大サイズ**: 100×100
- **デフォルト**: 8×8（標準チェス盤）

### ブロックの種類
- **白マス**: `WHITE_CONCRETE`（白いコンクリート）
- **黒マス**: `BLACK_CONCRETE`（黒いコンクリート）

### 配置ルール
- 各マス: 3×3ブロック
- パターン: (行 + 列) が偶数なら白、奇数なら黒
- 配置位置: プレイヤーの足元から開始

### 計算式
```
総ブロック数 = (サイズ × 3) × (サイズ × 3)

例: 8×8のチェス盤
  → (8 × 3) × (8 × 3) = 24 × 24 = 576ブロック
```

## エラーメッセージ

| エラー | 原因 | 対処法 |
|-------|------|--------|
| `エラー: サイズは5以上で指定してください` | サイズが5未満 | 5以上の数値を入力 |
| `エラー: サイズは100以下で指定してください` | サイズが100超過 | 100以下の数値を入力 |
| `エラー: 数値を入力してください（5〜100）` | 数値以外を入力 | 正しい数値を入力 |

## トラブルシューティング

### Q: プログラムが動かない
A: Pythonモードが選択されているか確認してください

### Q: チェス盤が表示されない
A: プレイヤーの足元に作成されます。下を見てください

### Q: ブロックが足りない
A: クリエイティブモードで実行してください

### Q: サイズが大きすぎて重い
A: 小さいサイズから試してください（10以下推奨）
